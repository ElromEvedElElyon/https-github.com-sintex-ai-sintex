# Sintex.AI with ChatGPT Integration 🚀

## Buy STBTCX Tokens Directly Through ChatGPT

This is the official Sintex.AI interface with integrated ChatGPT plugin support for purchasing STBTCX tokens without leaving the ChatGPT conversation.

## 🎯 Features

### ChatGPT Purchase Button
- **Location**: Fixed on the right side of the screen
- **Visual**: Green gradient button with pulsing animation
- **Badge**: "BETA" indicator showing experimental status

### Legal Disclaimer Modal
- **Comprehensive Terms**: Full legal disclaimer in Portuguese
- **Required Acceptance**: Users must check the agreement checkbox
- **Key Points**:
  - Purchase at your own risk
  - No refunds policy
  - Waiver of legal disputes
  - Beta product warning
  - Volatility disclaimer

### Integration Features
- Direct connection to ChatGPT's Agentic Checkout
- Session management for secure transactions
- Real-time price updates
- Transaction monitoring
- Automatic token delivery to Solana wallet

## 🚀 Deployment

1. **Clone the repository**:
   ```bash
   git clone https://github.com/StandardBitcoin10/Sintex.Ai-Build-Gemini-007.git
   ```

2. **Deploy to web server**:
   - Upload all files to your web server
   - Ensure HTTPS is enabled (required for ChatGPT)
   - Configure API endpoints

3. **Configure ChatGPT Plugin**:
   - Register at https://platform.openai.com/plugins
   - Submit manifest.json
   - Wait for approval

## 💻 Usage

1. **Click "Buy with ChatGPT"** button on the right
2. **Read and accept** the legal disclaimer
3. **Connect to ChatGPT** (opens in new window)
4. **Complete purchase** through conversation
5. **Receive tokens** automatically to wallet
